# RASAN AI Labs - AI-Powered AutoML Platform

RASAN AI Labs is an advanced AutoML platform that automates end-to-end machine learning workflows, enabling businesses and researchers to derive actionable insights from data without requiring deep technical expertise.

## Features

- **Automated Data Processing**: Intelligent handling of missing values, outliers, and feature engineering
- **Advanced Model Selection**: Automated selection of the best performing models for your data
- **Hyperparameter Optimization**: Automated tuning of model parameters for optimal performance
- **Model Interpretability**: Built-in SHAP values and feature importance analysis
- **Interactive Dashboard**: User-friendly interface for data exploration and model deployment
- **Export Capabilities**: Easy model export and deployment options

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/rasan-ai-labs.git
cd rasan-ai-labs
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
streamlit run app.py
```

## Usage

1. **Upload Data**: Start by uploading your dataset in CSV format
2. **Data Profiling**: Explore your data with automated EDA
3. **Model Training**: Select your target variable and let RASAN AI Labs find the best model
4. **Model Analysis**: Review model performance and feature importance
5. **Export**: Download your trained model for deployment

## Requirements

- Python 3.8+
- See requirements.txt for full list of dependencies
